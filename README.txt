== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `wp-music.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Add music content through the 'Music' menu in WordPress
1. Create a shortcode `[music]` to display the music(s) information. Shortcode attributes should be year, genre.
